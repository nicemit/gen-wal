Gen-Wal profiles support "Smart Prompts" using YAML Frontmatter. This creates self-contained "Mindset Packs" that define specific personas without needing global config.

Example: my_focus.md

---
quote_prompt_template: "Act as a Drill Sergeant. SCREAM the quote. Max 15 words."
image_prompt_template: "Generate a prompt for a gritty, dark industrial gym. Iron, shadows."
---
# Iron Mode
Strength, Pain, Victory.

To switch to this profile:

genwal profile use my_focus

Global Settings (config.yaml)

Gen-Wal uses a path-based configuration system.

  • quote_provider: "llm:ollama" -> Uses config from llm -> ollama.
  • quote_provider: "csv:work" -> Uses config from csv -> work.

Core Settings

Key Description Default
profile_path Path to your personal profile or list of profiles. profiles/examples/stoic.md
quote_provider Source for text generation. zenquotes
image_provider Source for background image. pollinations:image

Quote Providers

LLM (Local/Cloud)

quote_provider: "llm:ollama"

llm:
  ollama:
    base_url: "http://localhost:11434/v1" 
    api_key: "ollama" 
    model: "llama3.2"

Hugging Face

quote_provider: "huggingface:text"
huggingface:
  api_key: "hf_YOUR_TOKEN"

Image Providers

Pollinations.ai (Free, No Auth)

image_provider: "pollinations"
pollinations:
  model: "flux" 
  nologo: true

Watermark

Overlay a subtle text watermark on the wallpaper.

watermark:
  enabled: true
  text: "My Custom Watermark" # Optional
  position: "bottom_right"
  font_size: 25
  opacity: 150

CLI Overrides

Temporarily override settings at runtime:

Argument Description
--profile Use a specific profile file.
--text-pos Override text position.
--no-watermark Disable watermark for this run.
